La mejora actualmente implementada consiste en la aplicacion de un proceso de calibración y filtrado a las lecturas de los sensores. Primero, se realiza una calibración inicial calculando un offset a partir de las primeras muestras, lo que permite corregir posibles errores sistemáticos del sensor desde el inicio. Posteriormente, se aplica un filtro de media móvil, el cual suaviza las variaciones bruscas y reduce el ruido en las señales. Finalmente, se almacenan tanto los datos originales como los filtrados en el archivo CSV, lo que permite comparar los resultados y asegurar un análisis más preciso de la información obtenida.
